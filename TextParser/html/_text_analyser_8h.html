TextAnalyser.h

Das ist eine Bibliothek für die Analyse von Texten. Texte können aus Dateien geladen und bereinigt werden. Sie enthält Basisfunktionen wie Wortanzahl, Wortverteilung, Wort- oder Satzlänge. Zusätzlich kann man sich auch den Lesbarkeitswert berechnen lassen oder häufige vorkommen von wortpaaren ausgeben lassen
